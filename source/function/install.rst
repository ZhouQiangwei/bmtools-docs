Installation
=============

The Basic DNA Methylation (BM) format and BMtools

.. contents:: 
    :local:

Requirements
-------------

* gcc >= v4.8 
* :doc:`gsl`
* :doc:`zlib`

The details of requirements can see :doc:`Requirements`

Install
-------
.. code:: bash

    a) git clone https://github.com/ZhouQiangwei/bmtools.git
    b) Change directory into the top directory of bmtools
	$ cd bmtools
    c) Type 
	$ make
    e) The binary of bmtools will be created in ./

.. tip:: For feature requests or bug reports please open an issue `on github <http://github.com/ZhouQiangwei/bmtools>`__.
